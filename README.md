# Student Performance Prediction using Machine Learning

## Overview

This project uses Machine Learning to predict student final grades (G3) based on various features such as study time, previous grades, and family background.

## Dataset

- UCI Student Performance Dataset
- Includes:
  - Demographic data
  - Academic performance (G1, G2)
  - Study habits

## Algorithm Used

- Linear Regression

## Steps

1. Load dataset
2. Encode categorical data
3. Split into training and testing sets
4. Train model using Linear Regression
5. Predict student performance
6. Evaluate using Mean Squared Error (MSE)

## Results

- Mathematics Dataset MSE: ~5.03
- Portuguese Dataset MSE: ~1.43

## Conclusion

The model successfully predicts student performance with reasonable accuracy. It demonstrates how machine learning can be applied in education.
